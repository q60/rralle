rrallë
======

**Rrallë** (IPA: [raːɫ], stylized as *rrallë*) is a random quote
fetching console utility. Written in Erlang.

|screenshot|

Installing
----------

+ From AUR - `rralle <https://aur.archlinux.org/packages/rralle/>`_

Building and installing manually
--------------------------------

.. code-block:: sh

    make
    sudo make install

Running
-------

.. code-block:: sh

    rralle

Uninstalling
------------

.. code-block:: sh

    sudo make uninstall
